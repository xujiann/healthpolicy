机构分类依据

司局和处室口径按官方机构设置、三定规定及公开职能文件整理；补充政策支持人工归口，旧数据保留规则归口并可逐步校核。

平台数据治理

模拟卫生健康信息平台的数据接入、归口审核、知识图谱计算和公开链接追溯流程。

接入源

国务院政策库、国家卫健委、国家医保局、国家疾控局、国家中医药局等官方渠道。

归口规则

人工归口优先，规则归口进入审核清单；所有文件均落到部委、司局、处室层级。

自动任务

GitHub Actions 每日定时运行更新脚本，生成日志并在发现变化时提交入库。

追溯能力

每张卡片保留文件名、文号、摘要和官方全文链接，可导出 CSV 进行二次分析。

时间线

从顶层战略到专项制度，政策重心从“建框架”逐步转向“强体系、提质量、控费用、下基层”。

政策清单与官方链接

每条记录均为具体政策文件，链接指向国务院政策库或相关官方发布页面。

关于

本网站用于卫生健康政策文件的结构化归集、分类映射、历史演进趋势研究、未来发展方向预测和官方链接追溯。
